Annotate <audio>

the jQuery annotate plugin can add full HTML annotations to HTML5 <video> or <audio> elements.

It uses the HTML5 media elements API. It can be used for:

  • lyrics sing-alongs (woo!)
  • rich text callouts
  • captions
  • youtube-style video annotations (useful for video podcasts)

view-source or see this gist for the MIT-licensed source. [boston music hack day wiki project page]
